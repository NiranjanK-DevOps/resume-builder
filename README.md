# Resume Builder

A simple form-based resume generator hosted using AWS S3 Static Website Hosting.

## Features
- Users can enter name, email, phone, and skills
- Generates a clean resume layout instantly
- Fully client-side (HTML, CSS, JavaScript)
- No backend required
- Deployed on AWS S3 as a static website

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS S3
